I’d strongly recommend not using the transparent image just to avoid an unnecessary HTTP request. However, on the image topic, setting the background-image does seem like the best solution. You can still get this to work without splicing up your image (and without the additional HTTP request). Just use the same technique as you’d use for CSS sprites. Throw in the image that you’re already using for your background, and either position it so the images align, or just position it so it goes beyond the element (with no repeat flipped on).
