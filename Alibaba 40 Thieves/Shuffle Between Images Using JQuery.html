Shuffle Between Images Using JQuery

January 22nd, 2011 by aditia rahman / 2 Comments  

     

Shuffle mean changing the position between each items position, I continue this shuffling images feature so that can be added to my previous post about creating collage image gallery using jquery, and better can be added with rotating feature too. I’m using the same layout and images position like before.

The JQuery

The idea is when the page loaded, I store all the image source as a string in array, each tag have a unique id so it will make me easier to reset to the current position, and for the shuffling I just changing each image source by randomizing index in array

24/* this function from:
25 * http://www.admixweb.com/2010/08/24/javascript-tip-get-a-random-number-between-two-integers/
26 */
27function randomFromTo(from, to){
28    return Math.floor(Math.random() * (to - from + 1) + from);
29}

31function shuffle() {
32    var children = $("#container").children();
33    var child = $("#container div:first-child");
34 
35    var array_img = new Array();
36 
37    for (i=0; i<children.length; i++) {
38        array_img[i] = $("#"+child.attr("id")+" img").attr("src");
39        child = child.next();
40    }
41 
42    var child = $("#container div:first-child");
43 
44    for (z=0; z<children.length; z++) {
45        randIndex = randomFromTo(0, array_img.length - 1);
46 
47        // set new image
48        $("#"+child.attr("id")+" img").attr("src", array_img[randIndex]);
49        array_img.splice(randIndex, 1);
50 
51        child = child.next();
52    }
53}
